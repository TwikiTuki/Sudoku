## Backtracking Sudoku

I have never really gotten into the Sudoku world. The number of Sudokus I have started without ever finishing them is much higher than the ones I have finished. So, in order to try to compensate for this, I implemented an algorithm that solves Sudokus.

I have also created a basic interface to display some preloaded Sudokus and show their solutions. It is also possible to edit a Sudoku, and if you are committed enough, you can clear one and input a completely new one.

If you want to try it, you can [click here](https://twikituki.github.io/Sudoku/)

In case you want to run it locally, you will need a running server. One easy option to set it up is: ```python3 -m http.server```, but keep in mind that running a server this way is discouraged in production environments.
